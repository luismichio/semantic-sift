1. Introduction

Project X is a distributed system designed for high-throughput context distillation. It bridges the gap between raw data ingestion and semantic understanding.

2. Architecture Overview

The system consists of three main tiers: Ingestion, Distillation, and Delivery.

Tier Responsibility Technology
Ingestion Format conversion and normalization MarkItDown, Pandas
Distillation Prompt compression and ranking LLMLingua-2, BERT
Delivery MCP tool exposure FastMCP, Python

3. API Reference

The core API is exposed via Model Context Protocol (MCP). The primary entry point is the sift_read_file tool.

            GET /api/v1/sift
            {
                "path": "data/report.pdf",
                "rate": 0.5
            }

4. Security Considerations

All data is processed locally. Telemetry is anonymized and can be disabled via environment variables.
